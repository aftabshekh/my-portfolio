# 🚀 Aftab Shekh — Portfolio Website

A modern, responsive personal portfolio website built with React.js

## 🌐 Live Demo

[View Portfolio](https://my-portfolio-delta-eight-49.vercel.app) <!-- Vercel deploy hone ke baad update karna -->

## 🛠️ Built With

- **React.js** — Frontend Framework
- **Vite** — Build Tool
- **EmailJS** — Contact Form
- **CSS-in-JS** — Styling

## ✨ Features

- ✅ Responsive Design
- ✅ Smooth Scroll Navigation
- ✅ Working Contact Form (EmailJS)
- ✅ Resume Download
- ✅ Social Links
- ✅ Project Showcase
- ✅ Skills Section
- ✅ Animated UI

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/aftabshekh/my-portfolio.git

# Go to project folder
cd my-portfolio

# Install dependencies
npm install

# Run locally
npm run dev
```
